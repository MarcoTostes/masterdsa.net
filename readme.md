# masterdsa.net

masterdsa.net is a personal project built to provide clean, practical, and easy to follow resources on Java, Data Structures & Algorithms, and System Design. It focuses on transforming complex technical concepts into simple explanations, supported by real implementation examples and modern architectural practices. The platform is designed as a long term knowledge base that grows over time covering fundamentals, advanced topics, and hands on problem solving.

Behind the scenes, masterdsa.net is developed using a scalable and modern tech stack, ensuring high performance, reliability, and production grade architecture principles.

## Used Technologies

- **Microservices Architecture**
- **Kubernetes**
- **Docker**
- **Spring Boot**
- **Go**
- **Kafka**
- **Elastic Search**
- **Next.js**
- **Redis**
- **PostgreSQL**
- **MongoDB**
